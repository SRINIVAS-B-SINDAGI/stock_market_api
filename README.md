# Stock Market Information Backend

## Steps to run the project

Install the client project dependencies:

```bash
yarn install
```

To start in development environment

```bash
yarn dev
```

To start in prod environment

```bash
yarn start
```

Environement Variable to add in `.env file`

```
SERVER_PORT=
CLIENT_ORIGIN_URL=
AUTH0_AUDIENCE=
AUTH0_DOMAIN=
DB_NAME=
DB_CONNECTION_URL=
```
